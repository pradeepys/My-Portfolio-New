2. Smile to pay

  • Timeline : 3 weeks

Problem Statement


Is there any way to do payment process only by FACE without need of mobile in malls or stores at merchant place ?

If yes, then how does the user flow looks like, the merchant application and specific features to use it ?

How does the user will interact with the merchant/customer applications respectively ?

How does the user will track his/her payment history with customer application ?

Goal

Creating a mobile application design with all the features required to make payment process easier.

Making the User more comfortable with the application experience with a secure, trust and more efficient technology using Artificial Intelligence (AI).

UI/UX Process

Step 1 : Sketching
