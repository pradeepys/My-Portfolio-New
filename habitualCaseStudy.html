1. Habitual Shopping App

  • Timeline : 4 weeks

Problem Statement


Is there any way to do an online shopping via mobile application in an easy manner ?

If yes, then how does the user flow looks like, the application and specific features to use it ?

How does the user interacts the application with some cool animations ?

Goal

Creating a mobile application design with all the features required to make an online shopping.

Making the User more comfortable with the application experience to do an online shopping with efficient steps to complete the process.

UI/UX Process

Step 1 : Site Maps

Drawing the sitemaps, where all the gathered informations from the client that needs to be analysed all possible ways and drawing using any tool or on a paper and making a document for future purposes.
We used FIGMA tool for the project.

Step 2 : Sketching

Sketching the product how should the application looks like on a paper, which makes faster to do any changes. Once all the sketches are finalized, going to Step 3.

Step 3 : User Flows

User Flows helps to know how the user will interact with the product. Some of the user flows below,

Step 4 : Wireframes

Wireframes are creating a low fidelity design with the help of components like image, icons, buttons, text etc.

See all the wireframes here

Step 5 : Prototyping

Prototyping allow us to create interactive flows that explore how a user may interact with your designs with some cool micro animation. And also we should get the feedback and do necessary changes accordingly before going to step 6.

See some interations here in prototype tab

Step 6 : Visual Design

Visual Design is all about high fidelity design selecting the proper Color, Forms + UI Elements, Grids + Spacing, Imagenery + Iconography, Typography etc.

Making sure that the product is matching with the client requirements and handing over the design to the developers team to develop the application.

Successfully designed Habitual Shopping online mobile application with some specific features and cool animation effect.
